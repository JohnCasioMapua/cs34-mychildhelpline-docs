# How to Use the MyChild Helpline Application

## Introduction

MyChild Helpline is a mobile application that provides users with access to support services, educational resources, articles, workshops, and counseling-related activities. The application is designed to promote child welfare and provide users with information and assistance through an accessible digital platform.

## Prerequisites

* A compatible mobile device
* The MyChild Helpline application installed on the device
* An active internet connection for online features and updated content

## Getting Started

### Step 1: Open the Application

Locate the **MyChild Helpline** application on your device and tap the application icon to launch it.

### Step 2: Sign In or Continue as a Guest

If required, enter your login credentials to access your account. If guest access is available, select the appropriate option to proceed.

### Step 3: Navigate the Home Screen

The Home Screen provides access to the application's main features and services, including:

* Blog Articles
* MyChild Online Services
* Workshops and Educational Resources
* Counseling-Related Activities
* Account and Profile Settings

### Step 4: Access Available Features

Select the desired feature from the navigation menu and follow the instructions displayed on the screen.

### Step 5: Return to the Home Screen

Use the navigation controls or the back button to return to the Home Screen and access additional features.

## Expected Result

The user successfully navigates the MyChild Helpline application and accesses the desired information, services, and resources.

## Additional Notes

* An internet connection is required to access online services and updated content.
* New articles, workshops, and resources may be added periodically.
* Users should keep the application updated to access the latest features and improvements.

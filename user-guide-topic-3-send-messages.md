## Topic 3: Send Messages via Chat Helpline

You can send texts to a real agent using the Chat Helpline. Use this feature when you need immediate help, or want to ask something. 

### Before you begin

Make sure you have an active internet connection.

### Sending a message

1. Open the **MyChild Helpline** app.
2. Select **Chat Helpline** on the home screen.
3. Read the notice that appears.
4. Select **I Understand** to confirm and go to your chat screen.
5. Tap the **Message** field at the bottom of your screen.
6. Type your message.
7. Select **Send**.

A support agent reads your message and answers during active hours. A notification shows on your device when they reply.

### Ending a chat session

1. Select the **menu icon** at the top right of your chat screen.
2. Select **End Chat**.
3. Select **Confirm** when prompted.

The app saves a summary of your session. You can access it from **My Activity** on the home screen.

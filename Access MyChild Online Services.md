# Access MyChild Online Services

## Purpose

The MyChild Online Services feature provides users with access to additional resources and support services, including workshops, informational articles, counseling-related activities, and other educational content available through the MyChild Helpline platform.

## Prerequisites

* MyChild Helpline is installed on the user's device.
* The user has an active internet connection.
* The user has successfully accessed the application.

## Procedure

### Step 1: Open the Application

Launch the MyChild Helpline application on your device.

### Step 2: Access MyChild Online Services

From the application's main menu, select **MyChild Online Services**.

### Step 3: Browse Available Services

Review the list of available services, workshops, articles, and counseling-related activities.

### Step 4: Select a Service

Tap the desired service or resource to view additional information.

### Step 5: Access the Content

Read the selected article, view workshop details, or participate in the available counseling-related activity.

### Step 6: Return to the Services Menu

Use the navigation controls to return to the MyChild Online Services page and access other available resources.

## Expected Result

The selected service, article, workshop, or counseling-related resource is displayed successfully, allowing the user to access additional support and educational content.

## Troubleshooting

| Issue                                    | Solution                                                           |
| ---------------------------------------- | ------------------------------------------------------------------ |
| Services do not load                     | Verify that the device is connected to the internet and try again. |
| Unable to access MyChild Online Services | Restart the application and reopen the feature.                    |
| Content does not display correctly       | Refresh the page or reopen the selected service.                   |
| Application becomes unresponsive         | Close and relaunch the application.                                |

## Additional Notes

* Available services may vary depending on the user's location and supported country.
* New workshops, articles, and activities may be added periodically.
* Internet connectivity is required to access online resources and services.

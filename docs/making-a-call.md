# Making a call using  "Call Us" 

## Overview

The **Call Us** feature shows the national toll-free helpline number for your country and lets you dial it. Use this if you need immediate support.

## Making a Call

1. Open the **MyChild Helpline** app.
2. Select **Call Us** on the home screen.

The app shows the helpline number for your country.

3. Select the helpline number.

Your device's phone app opens with the number pre-dialed.

4. Select **Call** to connect.

The call connects you to a support agent at no charge.

> **Note:** Make sure your device can make calls before using **Call Us**. 

## If the Wrong Country Number Appears

The app shows your helpline number based on your location in your profile. If the number is for the wrong country:

1. Select **Settings** from the home screen.
2. Select **Profile**.
3. Update your **Location** to the correct country.
4. Select **Save Changes**.

Return to **Call Us** on the home screen. The correct number now appears.

# View My Rights

## Overview

The **My Rights** feature presents the **United Nations Convention on the Rights of the Child (UNCRC)** in a simple and interactive format. You can read about children's rights, listen to audio, and navigate through each right using swipe gestures.

![My Rights](../assets/my-rights.gif)

## Opening My Rights

1. Open the **MyChild Helpline** app.
2. Select **Information Kiosk** from the bottom navigation bar.
3. Select **My Rights**.

The app opens the **My Rights** feature.

## Viewing the Introduction

When you open **My Rights**, **Helping Hand** introduces the concept of children's rights.

### Listening to the Introduction

Select **Play** to hear the introduction.

* Select **Play** to start the audio.
* Select **Pause** to stop the audio.

## Viewing Children's Rights

To continue to the main content, select **Let's Go**.

The app displays one child right at a time. You can read the right, view its description, or listen to it using the audio controls.

There are 41 child rights available.

## Navigating Between Rights

* Swipe left to view the next child right.
* Swipe right to return to the previous child right.

The app updates the screen to display the selected child right.

## Returning to the Previous Screen

Select the **Back** (`<`) icon in the upper-left corner of the screen to return to the previous page.

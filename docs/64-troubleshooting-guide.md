# Troubleshooting Guide

## Overview

This guide provides solutions to common issues you may encounter while using the **MyChild Helpline** app. Review the issue, possible cause, and recommended solution below to help resolve the problem.

## Installation and Launch Issues

| **Issue**               | **Possible Cause**           | **Recommended Solution**                                                 |
| ----------------------- | ---------------------------- | ------------------------------------------------------------------------ |
| App fails to install    | Insufficient storage space   | Free up storage on your device and try installing the app again.         |
| Download is interrupted | Unstable internet connection | Check your internet connection and restart the download.                 |
| App does not open       | Outdated operating system    | Update your device to the latest supported version and relaunch the app. |
| Installation is stuck   | Temporary system error       | Restart your device and try installing the app again.                    |

## User Profile and Navigation Issues

| **Issue**                        | **Possible Cause**                       | **Recommended Solution**                                            |
| -------------------------------- | ---------------------------------------- | ------------------------------------------------------------------- |
| Unable to complete profile setup | Required fields are incomplete           | Fill in all required information before proceeding.                 |
| Selected language is incorrect   | Incorrect language selected during setup | Return to the language settings and choose your preferred language. |
| Cannot find a feature            | Unfamiliarity with the app interface     | Use the navigation menu or refer to the User Guide.                 |
| Buttons do not respond           | Temporary application issue              | Close and reopen the app, then try again.                           |

## Feature-Specific Issues

| **Issue**                              | **Possible Cause**                    | **Recommended Solution**                                                     |
| -------------------------------------- | ------------------------------------- | ---------------------------------------------------------------------------- |
| Unable to send a Chat Helpline message | No internet connection                | Connect to the internet and try sending the message again.                   |
| No response from a support agent       | Outside active support hours          | Wait for a response during active service hours and check for notifications. |
| Audio does not play in **My Rights**   | Device volume is muted or too low     | Increase your device volume and select **Play** again.                       |
| Unable to join a counseling session    | Session has not started or was missed | Verify your scheduled time and join a few minutes before the appointment.    |
| Cannot save a diary entry              | The save action was not completed     | Select the **Save** icon before leaving the page.                            |

## General Usage and Performance Issues

| **Issue**                   | **Possible Cause**       | **Recommended Solution**                            |
| --------------------------- | ------------------------ | --------------------------------------------------- |
| App loads slowly            | Poor internet connection | Connect to a stable network and try again.          |
| Content does not display    | Network or server issue  | Refresh the page or restart the application.        |
| App freezes unexpectedly    | Temporary software issue | Force close the app and reopen it.                  |
| Information is not updating | Cached or outdated data  | Restart the app or verify your internet connection. |

## Additional Support

If the issue persists after following the recommended solutions, restart your device and ensure you are using the latest version of the **MyChild Helpline** app. If the problem continues, contact the **MyChild Helpline** support team or consult the user documentation for further assistance.

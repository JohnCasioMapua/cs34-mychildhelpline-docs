# Send Messages via Chat Helpline

## Overview

The **Chat Helpline** feature allows you to send messages to a live support agent. Use this feature when you need immediate assistance or want to ask a question.

![Helping Hand](../assets/helping-hand.gif)

## Before You Begin

Make sure your device has an active internet connection.

## Starting a Chat

1. Open the **MyChild Helpline** app.
2. Select **Chat Helpline** on the home screen.
3. Read the notice displayed on the screen.
4. Select **I Understand** to continue to the chat screen.
5. Select the **Message** field at the bottom of the screen.
6. Type your message.
7. Select **Send**.

A support agent reviews your message and responds during active service hours. You will receive a notification when a reply is available.

## Ending a Chat Session

1. Select the **Menu** icon in the upper-right corner of the chat screen.
2. Select **End Chat**.
3. Select **Confirm** when prompted.

The app saves a summary of your conversation. You can view it later in **My Activity** on the home screen.

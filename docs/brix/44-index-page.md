# Index Page

## Overview

The Landing Page serves as the main dashboard of the MyChild Helpline App. It is the first screen users see after signing in and provides access to all major features of the app.

From this page, users can navigate to all sections of the app and return at any time.

## Profile display

### User profile section
At the top-center of the screen, the user’s avatar is displayed.

Below the avatar, the app shows a greeting message:

- **Good Morning, [Your Name]**

This section personalizes the user experience.

## Settings menu

### Opening settings
The Settings menu is located at the upper-left corner of the screen and is represented by a three-line icon.

Tap the icon to access the following options:

- Update your profile
- Enable diary lock
- Enable app lock
- Policies
- FAQs
- How to use the app
- Feedback
- Delete account
- About

## Accessibility and notifications

### Accessibility options
The Accessibility menu is located at the upper-right corner of the screen.

It includes the following features:

- Dyslexia Friendly
- Bigger text
- Line height adjustment
- Text spacing adjustment

### Broadcast messages
Broadcast messages are also available in the upper-right area of the screen and provide important updates and announcements.

## Main navigation categories

The bottom of the screen contains three main navigation tabs:

- My Space
- Information Kiosk
- Helping Hand

## My Space

### Features inside My Space
My Space provides personal support and well-being tools, including:

- Call us
- Request for counseling
- My diary
- My chill spot
- Arcade
- U-Matter Mental Health Textline

## Information Kiosk

### Features inside Information Kiosk
The Information Kiosk provides educational and support resources, including:

- My Rights
- Parenting Hub
- Government Services
- Resources
- MyChild Helpline
- UNICEF Eastern Caribbean Area

## Helping Hand

### Chat support
The Helping Hand section allows users to communicate with Helping Hand through a chat interface.

## Navigation controls

### Returning to the landing page
To return to the Landing Page from any screen:

- Tap the **“<” icon** located at the upper-left corner of the screen.

This ensures users can always navigate back to the main dashboard.

## Summary
The Landing Page is the central navigation hub of the MyChild Helpline App. It provides access to personal tools, informational resources, and support services, while also allowing users to customize accessibility and account settings.

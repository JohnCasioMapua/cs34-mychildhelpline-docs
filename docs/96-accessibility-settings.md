# Use Accessibility Settings

## Overview

The **Accessibility** settings allow you to customize the appearance of text in the **MyChild Helpline** app to improve readability. You can adjust **Dyslexia Friendly**, **Bigger Text**, **Line Height**, and **Text Spacing**.

## Opening Accessibility Settings

1. Open the **MyChild Helpline** app.
2. Select the **Accessibility** icon located beside the **Notification** (bell) icon in the upper-right corner of the screen.

The app displays the available accessibility options.

## Using Dyslexia Friendly

Select **Dyslexia Friendly** to cycle through the available settings.

* **First tap:** Applies the first level of dyslexia-friendly formatting.
* **Second tap:** Applies a higher level of dyslexia-friendly formatting.
* **Third tap:** Returns the setting to its default state.

## Adjusting Bigger Text

Select **Bigger Text** to cycle through the available text sizes.

* **First tap:** Increases the font size.
* **Second tap:** Applies a larger font size.
* **Third tap:** Returns the text to its default size.

## Adjusting Line Height

Select **Line Height** to cycle through the available spacing levels.

* **First tap:** Applies the first level of line spacing.
* **Second tap:** Applies the second level of line spacing.
* **Third tap:** Applies the third level of line spacing.
* **Fourth tap:** Returns the setting to its default state.

## Adjusting Text Spacing

Select **Text Spacing** to cycle through the available spacing levels.

* **First tap:** Applies the first level of text spacing.
* **Second tap:** Applies the second level of text spacing.
* **Third tap:** Applies the third level of text spacing.
* **Fourth tap:** Returns the setting to its default state.
